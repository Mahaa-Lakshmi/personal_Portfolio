RESEARCH PUBLICATIONS


Smart Vehicle Health Monitoring System Based On ARIMA and Ordinal Logistic Regression for Analysis of Vehicle Parameters

Conference: 2023 International Conference on System, Computation, Automation and Networking (ICSCAN)
Author(s): Mahaalakshmi B; Padmapriya R; Ellora Jeya Suji J; Cloudin S
Published in IEEE · Jan 26, 2024
Link: https://ieeexplore.ieee.org/abstract/document/10395495 View Paper

This study presents the development of a user-friendly web application designed to monitor vehicle health through the Internet of Vehicles (IoV). The IoV concept enables a network of vehicles to communicate and share data, providing real-time insights into their condition. Machine learning models, including Autoregressive Integrated Moving Average (ARIMA) and Ordinal Logistic Regression, were applied to predict future vehicle states. The system categorizes vehicle health into Good, Moderate, or Bad, assisting users in proactive vehicle maintenance and management.

Mental Health Prediction

This project introduces a Streamlit web application designed to predict an individual's risk of depression (high or low). It utilizes a deep learning model built with PyTorch, trained on mental health survey data encompassing lifestyle, demographic, and mental health-related responses.

The application facilitates predictions through two convenient input methods: users can either upload a CSV file containing survey data or manually enter information via a form. This ensures accessibility and ease of use for diverse user needs, leveraging extensive data preprocessing for accurate risk assessment.

SmartPremium Predictor

SmartPremium is an AI-powered system that uses machine learning to predict insurance premiums. It leverages Python, Scikit-Learn, and XGBoost to build predictive models, enhancing accuracy through advanced data processing and feature engineering. The project also incorporates Bayesian Optimization for model tuning, MLflow for managing the machine learning lifecycle, and Streamlit for the user interface, creating a complete and efficient pipeline.

Zomato Data Insights

The Zomato Food Delivery Data Insights project is a Python, MySQL, and Streamlit platform for food delivery data analysis and database management. It enables users to perform CRUD operations on key tables (Customers, Orders, Restaurants, Delivery Persons, Deliveries) and gain actionable insights on peak times, popular items, customer preferences, and delivery performance.
